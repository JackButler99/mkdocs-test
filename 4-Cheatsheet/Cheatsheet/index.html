Cheatsheet

Dokumen ini menyediakan ringkasan untuk hampir semua syntax elemen pada Markdown.

Elemen Syntax Link
Heading # h1 Headings
## h2
### h3
Bold **Bold Text** Bold and Italic (Emphasis)
Italic *italicized text* Bold and Italic (Emphasis)
Blockquote > blockquote Blockquotes
Code ```code``` Fenced Code Blocks
Horizontal Rule --- Horizontal Rule
Link [title](URL) Links
Image ![alt text](Image URL) Images
Footnote Sentence with footnote [^1] Footnotes
[^1]: The Footnote
Strikethrough ~\~\~ strikethrough ~\~\~ Strikethrough
